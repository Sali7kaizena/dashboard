TV Mode

TV Mode feature helps a user to display the charts during office presentations over television or
wallboard. Use the Play button to rotate the dashboard items at certain intervals of time.

Enhanced Connectivity

With our latest update, you can effortlessly fetch data from external PostgreSQL and MySQL databases by establishing secure connections and external API(Beta Version). Streamline your data access and make informed decisions with ease.
